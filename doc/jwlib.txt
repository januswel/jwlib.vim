*jwlib*	Autoload scripts by janus_wel

DESCRIPTION						*jwlib-description*

Library written by janus_wel.  This library provides various functions to
comfort you when write scripts and configure VIM.

For now, following files are available.

buf.vim
buf/replace.vim
	About buffer.  See |jwlib-buf.txt| for detail.

color/rgb.vim
color/xterm256.vim
	About color.  See |jwlib-color.txt| for detail.

fs.vim
	About filesystem.  See |jwlib-fs.txt| for detail.

shell.vim
	About shell.  See |jwlib-shell.txt| for detail.

tabline.vim
	About tabline.  See |jwlib-tabline.txt| for detail.

unicode.vim
unicode/namelist.vim
	About Unicode.  See |jwlib-unicode.txt| for detail.
